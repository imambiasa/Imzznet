# imzznet - YouTube Thumbnail Downloader

Website untuk download thumbnail YouTube dengan berbagai kualitas HD.

## Fitur

- Download thumbnail YouTube dalam 5 kualitas berbeda
- Preview thumbnail sebelum download
- Responsive design
- 100% gratis

## Cara Menggunakan

1. Masukkan URL video YouTube
2. Klik "Generate Thumbnail"
3. Pilih kualitas yang diinginkan
4. Download thumbnail
